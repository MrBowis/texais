# texais
A full complete tool to handle pdf and latex with AI

## Frontend

start frontend:
```bash
cd ./src/frontend/texais-frontend
ng serve
```

## Recomendations

Create a virtual environment:
```bash
python3 -m venv venv
source venv/bin/activate
```

Install requirements:
```bash
pip install -r requirements.txt
```
